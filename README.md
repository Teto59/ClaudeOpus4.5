# JobTracker - 就活管理アプリ

就職活動の進捗を一元管理できるWebアプリです。

## 機能

- **企業管理**: 企業の追加・編集・削除、ステータス管理
- **締切管理**: ES/Webテストの締切や面接日程を一覧表示
- **ステータス追跡**: ES/Webテスト → 一次面接 → 二次面接 → 三次面接 → 内定
- **カスタムステータス**: 自分でステータスを追加可能
- **企業メモ**: 各企業ごとにESの内容などをメモ
- **全体メモ帳**: 自己PR、ガクチカなどのテンプレートを管理

## 特徴

- **ダークモード**: 夜にも見やすいデザイン
- **プライバシー保護**: データはブラウザのローカルストレージに保存（外部には送信されません）
- **設定不要**: アカウント登録なしですぐに使える

## 使い方

1. 「企業追加」ボタンで企業を登録
2. 企業をクリックして詳細画面でメモを記入
3. ステータスや締切を更新
4. 「メモ帳」タブで汎用的なテンプレートを管理

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
